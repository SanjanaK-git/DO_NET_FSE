# Lab 1 - Understanding ORM with a Retail Inventory System

## Objective
Understand Object Relational Mapping (ORM) and Entity Framework Core.

### What is ORM?
ORM maps C# classes to relational database tables.

### Benefits
- Productivity
- Maintainability
- Database abstraction
- Less SQL coding

### EF Core vs EF Framework
- EF Core is cross-platform.
- EF Framework works only on Windows.
- EF Core is lightweight and faster.

### EF Core 8.0 Features
- JSON Column Mapping
- Compiled Models
- Better Performance
- Interceptors
- Bulk Operations
